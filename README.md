# TXRJets-Charter-Request
Official Request Quote System app for private aviation booking, crew management, and operational tracking by TenX Reserved Jets.

Welcome to the official mobile gateway for TenX Reserved Jets booking platform.

## Features
• **Real-time Quotes:** Instant pricing for Light, Mid-Size, and Heavy Jets.
• **Flight Tracking:** Monitor your chartered missions in real-time.
• **Aircraft Availability:** Booking marked confirmed when an aircraft have been secured for your flight.

## Installation
1. Download the `txrjets_charter_request_v1.apk` file from the **Releases** section.
2. Open the file on your Android device.
3. If prompted, allow "Install from Unknown Sources" in your settings.
